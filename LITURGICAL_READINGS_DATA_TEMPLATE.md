# Liturgical Readings Data Template

## Overview

This document provides sample reading data for the `LiturgicalReadings` sheet. Use this as a reference for data entry or import.

## Sheet Structure

**Sheet Name:** `LiturgicalReadings`

**Column Headers (Row 1):**
```
A: Liturgical Celebration
B: Cycle
C: Lectionary Number
D: Reading Type
E: First Reading
F: Responsorial Psalm
G: Second Reading
H: Gospel Acclamation
I: Gospel
J: Notes
```

## How to Use This Template

### Option 1: Manual Entry into Google Sheets

1. Create a new sheet named "LiturgicalReadings" in your parish scheduler spreadsheet
2. Copy the column headers into Row 1
3. Start entering data row by row from the samples above
4. Expand with additional celebrations as needed

### Option 2: CSV Import

1. Convert the tables above to CSV format:
   ```csv
   Liturgical Celebration,Cycle,Lectionary Number,Reading Type,First Reading,Responsorial Psalm,Second Reading,Gospel Acclamation,Gospel,Notes
   1st Sunday of Advent,A,1,Standard,Isaiah 2:1-5,Psalm 122:1-2; 3-4; 4-5; 6-7; 8-9,Romans 13:11-14,Psalm 85:8,Matthew 24:37-44,
   ```
2. Import into Google Sheets: File → Import → Upload CSV → Replace current sheet

### Option 3: Copy-Paste from Web Sources

Many liturgical websites (e.g., USCCB.org) display reading references. You can:
1. Navigate to the readings page for a specific date
2. Copy the reading references
3. Paste into your spreadsheet
4. Clean up formatting

## Data Coverage Strategy

### Minimal Coverage (Start Here)
- **Sundays for current liturgical year only** (~52 rows)
- **Major solemnities** (Christmas, Easter, Assumption, All Saints - ~15 rows)
- **Total:** ~70 rows
- **Time to enter:** 3-4 hours

### Standard Coverage
- **All Sundays (Year A, B, C)** (~156 rows)
- **Major solemnities** (~20 rows)
- **Holy Week and Triduum** (~8 rows)
- **Total:** ~185 rows
- **Time to enter:** 6-8 hours

### Comprehensive Coverage
- **All Sundays (Year A, B, C)** (~156 rows)
- **All solemnities and feasts** (~50 rows)
- **Weekday cycles (Year I and II)** (~600 rows)
- **Total:** ~800 rows
- **Time to enter:** 20-30 hours (but provides complete multi-year coverage)

## Data Sources

### Official Sources

1. **USCCB Daily Readings**
   - URL: https://bible.usccb.org/daily-bible-reading
   - Navigate by date to see reading references
   - Official USA lectionary

2. **Universalis**
   - URL: https://universalis.com/
   - Shows readings for any date
   - Includes full text (copyright restrictions apply)

3. **iBreviary App**
   - Mobile app with complete lectionary
   - May have export functionality

### Community Sources

1. **GitHub Repositories**
   - Search for "catholic lectionary csv" or "liturgical readings data"
   - Many parishes have shared their data

2. **Parish Liturgy Software**
   - Check if your existing software exports lectionary data
   - Common formats: CSV, Excel, XML

## Validation Checklist

After entering data, verify:

- [ ] All celebration names match those in LiturgicalCalendar sheet EXACTLY
- [ ] Cycle values are only: A, B, C, I, II, or Fixed
- [ ] Lectionary Number is populated for every row and is numeric
- [ ] Reading Type is populated (Standard, Vigil, Day, OT Reading 1-7, etc.)
- [ ] No duplicate celebration + cycle + reading type combinations
- [ ] At minimum, First Reading and Gospel are populated for every row (except Easter Vigil OT readings)
- [ ] Second Reading is blank for weekdays (correct - only Sundays/solemnities have 2nd reading)
- [ ] Run data validation: Admin Tools → Validate Data
- [ ] Test with print schedule: Generate schedule → Verify readings and lectionary numbers appear

## Tips for Efficient Data Entry

1. **Start with current liturgical year only** - You don't need all 3 Sunday cycles immediately
2. **Use copy-paste for Psalm responses** - Many repeat throughout the year
3. **Look for patterns** - Gospel Acclamations often repeat within seasons
4. **Batch entry by season** - All of Advent, then all of Christmas, etc.
5. **Use USCCB website** - Navigate to specific date, copy reading references
6. **Leverage AI** - Ask Claude or ChatGPT to format specific dates into CSV rows
7. **Share the work** - Multiple volunteers can work on different seasons simultaneously

## Future Enhancements

Once basic reading references are in place, consider:

1. **Full reading text** - Store complete readings for lector practice (large undertaking)

3. **Lector notes** - Add interpretive notes or emphasis suggestions
4. **Alternative readings** - When multiple options exist, track which your parish uses
5. **Multilingual** - Spanish, Vietnamese, or other language translations
